Custom Image Dice

Create dice with custom images for sides and create card decks from images

App description

Simple app that allows you to create custom dice with your images on dice sides and then roll these dice.

Main idea of this app is to help boardgamers to try out print and play games, that require custom dice to play. Alternatively this app can be used as replacement for special dice found in many boardgames, or as simple image randomizer.

App features:
  • Roll dice sets with dice locking feature
  • Creating dice sets
  • Creating custom dice with images on dice sides
  • Importing/exporting dice sets for easier sharing
  • Copying dice sets and dice for easier creation of new dice sets
